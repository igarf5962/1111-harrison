This is the gh-page for Physics 1111 with Dr. Harrison. Below is an area for running SageMath scripts. The default example shows how to solve the system of equations: $$ x + y = 0 \\ x - 2y = 6. $$ See below for more examples.



To solve the equation $$ \frac{6}{x+2} + \frac{2}{x-4} = \frac{-7}{x^2 - 2x - 8} $$ run the following script: 

x = var("x")
solve([6/(x+2) + 2/(x-4) == -7/(x^2 - 2*x - 8)], x)
or equivalently: 

x, y1, y2 = var("x, y1, y2")
y1 = 6/(x+2) + 2/(x-4)
y2 = -7/(x^2 - 2*x - 8)
solve([y1 == y2], x)
